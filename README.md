# Planetoño - The Space Food Station 🪐

Welcome to the **Planetoño** landing page! This is a highly interactive, scroll-driven, 3D and WebGL-powered web experience designed to showcase an immersive, premium, and futuristic food service concept.

## 🚀 Live Demo

Check out the live interactive experience here: **[The Drawn Dimension](https://the-drawn-dimenstion.sunilnathyogi008.workers.dev/)**

## ✨ Key Features

- **Immersive 3D & WebGL Graphics**: Rich interactive 3D elements powered by modern web technologies.
- **Scroll-Driven Storytelling**: Seamlessly navigate through the experience using smooth, physics-based scrolling animations.
- **Rive & Lottie Animations**: Vector-based micro-interactions that bring the UI to life with zero performance loss.
- **Dynamic Sound Design**: Built-in sound effects (`hover`, `click`, etc.) synchronized with interactions.
- **Fully Responsive**: A fluid design that adapts and scales beautifully across mobile, tablet, and desktop devices.
- **Premium UI/UX Aesthetics**: Creamy, vibrant styling (`#F7F2E5`) with custom typography and SVGs, presenting a state-of-the-art interface.
